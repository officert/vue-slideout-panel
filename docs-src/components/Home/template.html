vue-slideout-panel

Vue.js component for creating stackable slideout panels.

View on GitHub Download .zip Download .tar.gz

Installation

npm install vue-slideout-panel --save-dev

1) Download from NPM

Usage

1) Register the Vue Slideout Panel Component: 

import Vue from 'vue';
import { VueSlideoutPanel } 'vue-slideout-panel';

Vue.component('slideout-panel', VueSlideoutPanel);

2) Add the Vue Slideout Panel Component tag to your HTML: 

<slideout-panel></slideout-panel>

3) Use the Vue Slideout Panel Service to launch a panel: 

import { vueSlideoutPanelService } from 'vue-slideout-panel';

vueSlideoutPanelService.show({
  component : 'your-component-name',
  props: {
    //any data you want passed to your component
  }
})
.then(result => {

});

Let's say we have a simple page component called 'home' and we want to lauch a panel from it: 

import Vue from 'vue';
import { vueSlideoutPanelService } 'vue-slideout-panel';

export default {
  name: 'home',
  data() {
    return {}
  },
  methods: {
    showPanel1() {
      vueSlideoutPanelService.show({
          component: 'panel-1',
          props: {
            data: {
              firstName: 'John',
              lastName: 'Doe',
              age: 39
            }
          }
        })
        .then(results => {
          console.log('Results for panel 1:', results);
        });
    }
  }
};

Here's the compoent that we want to live inside the panel: 

export default {
  name: 'panel-1',
  props: {
    data: {
      type: Object,
      required: true
    }
  },
  data() {
    return {};
  },
  methods: {
    closePanel() {
      this.$emit('closePanel', {
        name: this.form.name
      });
    }
  }
};
